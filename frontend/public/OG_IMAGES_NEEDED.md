## Design Guidelines

1. **Branding**:
   - Use your existing dark theme (#111113 background)
   - Include API Stress Lab logo
   - Use emerald accent color (#10b981 or similar)
   - Consistent typography (Inter font family)

2. **Content**:
   - Clear, readable headline
   - Minimal text (people see these as small previews)
   - High contrast for readability
   - No important text near edges (may get cropped)

3. **Technical**:
   - Format: PNG or JPG
   - File size: Keep under 1MB (ideally under 300KB)
   - Color space: RGB
   - Resolution: 72 DPI minimum
